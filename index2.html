Principiante

Rutina

Nivel 1: Principiante (Full Body / Cuerpo Completo) Ideal para crear una base sólida y aprender los movimientos. Entrena 3 días a la semana (ej. Lunes, Miércoles y Viernes).

Rutina A (Enfoque Empuje):

    * Sentadillas con barra: 3 x 10.
    * Press de banca (en el banco Fuxion): 3 x 10.
    * Press militar con mancuernas: 3 x 12.
    * Plancha abdominal: 3 x 45 seg.

Rutina B (Enfoque Tracción):

    * Peso muerto rumano con barra: 3 x 10.
    * Remo con barra inclinado: 3 x 10.
    * Curl de bíceps con barra: 3 x 12.
    * Uso de hand grip: 3 x máximo esfuerzo.

Rutina C (Híbrida):

    * Zancadas (lunges) con mancuernas: 3 x 12.
    * Aperturas de pecho con mancuernas: 3 x 12.
    * Elevaciones laterales: 3 x 15.
    * Crunch abdominal en banco: 3 x 20
